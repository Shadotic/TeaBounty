# Contributing Guidelines

Welcome to the TeaBounty project! We appreciate your interest in contributing. Before you get started, please take a moment to review the following guidelines to ensure a smooth and productive collaboration process.

## How to Contribute

1. **Fork the Repository:** Start by forking the TeaBounty repository to your GitHub account.

2. **Clone the Repository:** Clone the forked repository to your local machine using Git.

    ```bash
    git clone https://github.com/your-username/TeaBounty.git
    ```

3. **Create a Branch:** Create a new branch for your contributions.

    ```bash
    git checkout -b feature/new-feature
    ```

4. **Make Changes:** Make your desired changes to the codebase. Ensure that your changes adhere to the project's coding standards and conventions.

5. **Test Your Changes:** Test your changes thoroughly to ensure they work as expected and do not introduce any regressions.

6. **Commit Your Changes:** Commit your changes with clear and descriptive commit messages.

    ```bash
    git commit -m "Add new feature"
    ```

7. **Push Your Changes:** Push your changes to your forked repository.

    ```bash
    git push origin feature/new-feature
    ```

8. **Create a Pull Request:** Once you have pushed your changes to your forked repository, create a pull request (PR) to merge your changes into the main TeaBounty repository. Provide a detailed description of your changes in the PR.

## Code Review

All contributions to TeaBounty will go through a code review process. Our maintainers will review your code and provide feedback. Please be patient during this process and be open to making any necessary changes to your code.

## Reporting Issues

If you encounter any bugs, issues, or have suggestions for improvements, please report them by opening an issue in the TeaBounty repository. Provide as much detail as possible, including steps to reproduce the issue and any relevant information about your environment.

## Community Guidelines

1. **Be Respectful:** Treat others with respect, kindness, and empathy. We welcome diverse perspectives and constructive criticism, but do not tolerate harassment, discrimination, or disrespectful behavior.

2. **Follow the Code of Conduct:** All contributors are expected to adhere to the TeaBounty [Code of Conduct](./CODE_OF_CONDUCT.md). Violations of the code of conduct will not be tolerated.

3. **Collaborate and Communicate:** Work together with other contributors in a spirit of collaboration and cooperation. Communicate openly and courteously, and be considerate of others' opinions and ideas.

4. **Have Fun:** Above all, have fun and enjoy your experience contributing to TeaBounty! We value your contributions and appreciate your efforts to make the project better.

Thank you for your interest in contributing to TeaBounty. Your contributions are highly valued and appreciated!

```
